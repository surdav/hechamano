# Tailwind build

Run from the repository root after changing Tailwind classes in `index.html`:

```bash
npx --yes tailwindcss@3.4.17 -i empresa_de_reformas_integrales_en_lloret_de_mar/src/input.css -o empresa_de_reformas_integrales_en_lloret_de_mar/assets/tailwind.css --content index.html --minify
```
